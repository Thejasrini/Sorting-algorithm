# Sorting Algorithm Visualizer

A simple and interactive **Sorting Algorithm Visualizer** built using **HTML, CSS, and JavaScript**.  
This project helps in understanding how different sorting algorithms work through **numbers-only visualization**.

---

## Project Description

Sorting algorithms are an important part of **Data Structures and Algorithms (DSA)**.  
This application visually demonstrates the sorting process by showing comparisons, swaps, and the final sorted output step by step.

The project is designed to be clean, easy to use, and educational.

---

## Features

- Numbers-only sorting visualization
- Algorithm selection
- Adjustable array size
- Speed control
- Start and stop sorting
- Color-coded steps for better understanding
- Displays time and space complexity

---

## Algorithms Implemented

- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Quick Sort  
- Merge Sort  
- Heap Sort  
- Counting Sort  
- Radix Sort  

Each algorithm is implemented in a separate JavaScript file.
